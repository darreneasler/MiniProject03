# MiniProject03

This project takes the used car data from **car details v4.csv ** and predicts a fair market price for used cars using a Ridge Regression. The error for this model is given as:

MAE: 401629.91

RMSE: 809180.04

R²: 0.832 

The notebook has comments describing the function of each cell. The PDF contain a deeper analysis of some of the deeper engineering choices made in training the model.
